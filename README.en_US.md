### Introduction
A mini-game of "Blue Flag, White Flag" using pose recognition. <br/>
This project was created for academic department promotion.

### Key Features
- Detecting raised hands using Mediapipe Pose
- Creating ROI (Region of Interest) above wrists to detect colors
- Playing the Blue Flag White Flag mini-game using a webcam
- Using pygame module to play sounds asynchronously
