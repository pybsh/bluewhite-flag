### 소개
포즈 인식을 이용한 청기백기 미니게임입니다. <br/>
학과 홍보를 위한 프로젝트입니다.

### 주요 기능
- Mediapipe Pose를 이용해 손을 들었는지 확인하기
- 손목 위 ROI를 생성해 색상 감지하기
- 웹캠을 이용한 청기백기 미니게임 진행
- 비동기적으로 소리를 재생하기 위해 pygame 모듈을 사용함
